DATA DELETION REQUEST – TRENDVIBE APPAREL STORE

This page explains how users of the TrendVibe Apparel Store app can request deletion of their personal data.

──────────────────────────────

HOW TO REQUEST DATA DELETION

Users can request deletion of their personal data by contacting us directly via email.

To submit a request:

1. Send an email to: keizahemmanuel4@gmail.com  
2. Use the subject line: “Data Deletion Request – TrendVibe App”  
3. Include your account email or any details used within the app  

──────────────────────────────

DATA THAT MAY BE DELETED

Upon request, we may delete the following user data:

- Name  
- Email address  
- Shipping address  
- Phone number (if provided)  
- Order-related information (if applicable)  

──────────────────────────────

DATA RETENTION

Some data may be retained for a limited period if required for:

- Legal obligations  
- Fraud prevention  
- Financial or transaction record keeping  

After the required retention period, data will be securely deleted.

──────────────────────────────

PROCESSING TIME

We aim to process all data deletion requests within a reasonable timeframe, in accordance with applicable laws and regulations.
